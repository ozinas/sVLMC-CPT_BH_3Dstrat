# sVLMC-CPT_BH_3Dstrat
This set of python notebooks and functions contain the code to reproduce some results of the paper: Zinas, Orestis and Papaioannou, Iason and Schneider, Ronald and Cu√©llar, Pablo, Multivariate Gaussian Process Regression for 3d Site Characterization from CPT and Categorical Borehole Data. Available at SSRN: https://ssrn.com/abstract=4995360 or http://dx.doi.org/10.2139/ssrn.4995360

This repository contains Jupyter notebooks utilizing **GPyTorch** for Gaussian Process modeling. The code is intended to **reproduce specific results** from our study and is not provided as a general-purpose model. It includes working scripts that were used to generate the reported results.
